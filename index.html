Project

TEAM COMPOSITION OF LEAUGE OF LEGENDS – INTERACTIVE VISUALIZATION

· Created an interactive tool that helps the online game, League of Legends players draft winning champion combination to improve their game. The team combination was calculated based on association rules and community detection algorithm. More than one gigabyte sized data was collected from Riot Games API. The first graph shows strong combinations of two champions and the second interactive graph helps with selecting strong communities of champion determined by detection.

RANDOM FOREST CLASSIFIER

· Developed Decision to perform binary classification on dataset derived from the National Institute of Diabetes to determine if a patient has diabetes. Implemented Random Forest machine learning algorithm using the decision trees to train the classifier. The performance of the classifier is evaluated accurately via the out-of-bag (OOB) error estimate.

CO-ACTOR NETWORK

· Created an interactive visualization of co-actor network which show which actors have acted together in different movies. Data was collected from The Movie Data Base using TMDb API version 3. Implemented graph with nodes and edges and stored them as csv files to visualize the network of actors and their co-actors using Argo Lite, an interactive graph visualization tool.
