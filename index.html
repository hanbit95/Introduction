Project

OPTIMAL TEAM COMPOSITION FOR LEAUGE OF LEGENDS

· Worked in a group of four to develop a user-interactive visualization for the League of Legends players to improve their win rate by optimizing a team formation. Collected more than one gigabyte worth of match data from Riot Games API to calculate effective champion combinations using association rules and community detection algorithm. Results were displayed as two graphs: one that shows each champion’s best pairing option and the other showing overall synergies among the champions.

RANDOM FOREST CLASSIFIER

· Built a decision tree model that performs a binary classification on the dataset from the National Institute of Diabetes to predict if a patient is likely to have diabetes. The model was then used to implement Random Forest machine learning algorithm, which combines multiple outcomes of randomly created decision trees to produce a final outcome. The accuracy of the classifier was evaluated using the out-of-bag(OOB) error estimate model.

CO-ACTOR NETWORK

· Created an interactive visualization that displays a network of movie actors and their co-actors. The visualization indicates the strength of each relationship between two actors based on how frequently they have been in movies together. Used TMDb API to extract data from The Movie Data Base, and stored graph’s nodes and edges as CSV files. Utilized Argo Lite to build and publish visualization.
